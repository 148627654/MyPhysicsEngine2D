# Day 03: 形状定义与转动惯量 (Shape & Inertia)

## 1. 目标
实现 `Shape` 类及其派生类（`Circle` 和 `Box`），并为 `RigidBody` 增加**旋转动力学**支持。

## 2. 核心概念：转动惯量 (Inertia)
*   **质量 (Mass)** 是对“平移运动”改变的阻碍。
*   **转动惯量 (Inertia)** 是对“旋转运动”改变的阻碍。
*   在物理引擎中，我们需要根据物体的形状和质量，预先计算出它的转动惯量 $I$。

- [ ] **角度属性**:

- [ ] **惯量属性**:
    - `float inertia`: 转动惯量。
    - `float invInertia`: 转动惯量的倒数（如果是静态物体，则为 0）。
- [ ] **接口更新**:
    - `AddTorque(float t)`: 累加转矩。
    - `SetShape(Shape* s, float density)`: 绑定形状并自动计算质量属性。

### C. 质量与惯量计算公式
在 `Shape` 的子类中实现以下逻辑：

| 形状              | 质量 (Mass)                  | 转动惯量 (Inertia)                 |
| :-------------- | :------------------------- | :----------------------------- |
| **圆形 (Circle)** | $m = \rho \cdot \pi r^2$   | $I = \frac{1}{2}mr^2$          |
| **矩形 (Box)**    | $m = \rho \cdot w \cdot h$ | $I = \frac{1}{12}m(w^2 + h^2)$ |

## 4. 技术实现细节 (Tips)

### 1. 动力学积分器升级 (World.cpp)
在 `World::Step` 中，除了更新位置，现在也要更新角度了：
```cpp
// 1. 角加速度 alpha = torque / I
float angularAcceleration = body->torque * body->invInertia;

// 2. 积分角速度
body->angularVelocity += angularAcceleration * dt;

// 3. 积分角度
body->rotation += body->angularVelocity * dt;

// 4. 清空转矩
body->torque = 0.0f;
```

### 2. 绑定形状的逻辑
建议在 `Body` 类中写一个辅助函数：
```cpp
void Body::SetShape(Shape* shape, float density) {
    // 假设是矩形
    this->mass = density * shape->GetArea();
    this->invMass = (mass > 0) ? 1.0f / mass : 0.0f;
    
    this->inertia = shape->CalculateInertia(mass);
    this->invInertia = (inertia > 0) ? 1.0f / inertia : 0.0f;
}
```

## 5. 今日验收测试 (Unit Test)

今天没有碰撞，但我们要验证**转动动力学**是否正确：

1.  **惯量验证**：
    - 创建一个质量为 `10kg`，宽高为 `2m x 2m` 的矩形。
    - 计算其 `inertia` 是否等于 `10 * (2^2 + 2^2) / 12 = 6.666...`。
2.  **转动验证**：
    - 给这个矩形施加一个恒定的转矩 `AddTorque(10.0f)`。
    - 运行 `Step(1.0/60.0)`。
    - 观察角速度 `angularVelocity` 是否在均匀增加，角度 `rotation` 是否加速改变。
3.  **静态验证**：
    - 将 `invInertia` 设为 `0`，施加转矩，确认物体不会旋转。
