# Day 10: 旋转冲量与偏心碰撞响应

## 1. 目标
升级冲量解算器，使其考虑**碰撞点相对于质心的偏移**。实现碰撞瞬间平动速度（Velocity）与角速度（Angular Velocity）的耦合更新。

## 2. 核心概念：偏心碰撞 (Eccentric Collision)
当力作用在物体的质心以外时，它会同时改变物体的平动和转动。

*   **接触点速度 ($\vec{v}_p$)**：物体在接触点 $P$ 处的总速度。
    $$\vec{v}_p = \vec{v}_{linear} + \omega \times \vec{r}$$
    其中 $\vec{r}$ 是从质心指向接触点的向量。
*   **旋转项的冲量公式**：
    由于引入了旋转，冲量标量 $j$ 的分母需要增加旋转抗性项：
    $$j = \frac{-(1+e)(\vec{v}_{rel} \cdot \vec{n})}{\frac{1}{m_A} + \frac{1}{m_B} + \frac{(\vec{r}_A \times \vec{n})^2}{I_A} + \frac{(\vec{r}_B \times \vec{n})^2}{I_B}}$$
    *(注：$I$ 是转动惯量，$r \times n$ 在 2D 中是一个标量)*

## 4. 技术实现细节 (Tips)

### 1. 2D 叉积的捷径
在 2D 物理中，叉积简化为：
*   `Vector2(x, y) × Scalar(w)` = `Vector2(w * y, -w * x)`
*   `Scalar(w) × Vector2(x, y)` = `Vector2(-w * y, w * x)`
*   `Vector2(x1, y1) × Vector2(x2, y2)` = `x1 * y2 - y1 * x2`

### 2. 接触点的选取
对于 Box-Box 碰撞，可能有多个接触点。**今天为了简化**，你可以直接取 `Manifold.contacts` 列表中的第一个点，或者取它们的平均值。

### 3. 这里的 IA 和 IB 是什么？
这就是你在 **Day 03** 算好的转动惯量。如果 `invInertia == 0`（静态物体），旋转项会自动消失。

### 💡 核心代码框架预览 (Solver.cpp)

```cpp
void ImpulseSolver(Manifold& m) {
    Body* A = m.bodyA;
    Body* B = m.bodyB;
    
    float invMassSum = A->getInvMass() + B->getInvMass();
    if (invMassSum < 0.0001f) return;

    // 假设只取第一个接触点
    Vector2 contact = m.contacts[0];
    Vector2 rA = contact - A->GetPosition();
    Vector2 rB = contact - B->GetPosition();

    // 接触点速度: v + w x r
    Vector2 vA = A->GetVelocity() + Vector2::Cross(A->angularVelocity, rA);
    Vector2 vB = B->GetVelocity() + Vector2::Cross(B->angularVelocity, rB);

    Vector2 vRel = vB - vA;
    float vNormal = vRel.Dot(m.normal);
    if (vNormal > 0) return;

    // 分母加入旋转项
    float rAn = Vector2::Cross(rA, m.normal);
    float rBn = Vector2::Cross(rB, m.normal);
    float rotTerm = (rAn * rAn * A->getInvInertia()) + (rBn * rBn * B->getInvInertia());

    float e = std::min(A->restitution, B->restitution);
    float j = -(1 + e) * vNormal / (invMassSum + rotTerm);

    Vector2 impulse = m.normal * j;

    // 同时更新平动和转动
    A->ApplyImpulse(impulse * -1.0f, rA); // 注意：需要升级 ApplyImpulse
    B->ApplyImpulse(impulse, rB);
}
```
