# Day 01: 向量数学库 (Vector2) 的设计与实现

## 1. 目标
构建一个高性能、健壮的 2D 向量类 `Vec2`。这是整个物理引擎的“度量衡”，几乎所有的后续计算（速度、力、位置、碰撞法线）都依赖于它。

## 2. 核心成员变量
使用 `float` 类型以平衡精度与性能：
- `float x;`
- `float y;`

## 3. 必须实现的功能清单
### A. 基础运算符重载 (Operator Overloading)
为了让代码像写数学公式一样自然：
- [ ] **加法/减法**: `+`, `-`, `+=`, `-=` (向量与向量)
- [ ] **标量乘除**: `*`, `/`, `*=`, `/=` (向量与 float)
- [ ] **取反**: `-Vec2` (单目运算符)
- [ ] **相等判断**: `==`, `!=` (注意：浮点数比较需引入 `EPSILON`)

### B. 几何运算 (Geometric Operations)
- [ ] **长度计算**: 
    - `Length()`: 真正的长度 $\sqrt{x^2 + y^2}$。
    - `LengthSquared()`: 长度的平方 $x^2 + y^2$ (性能优化关键，能不用开方就不用)。
- [ ] **归一化 (Normalization)**: 
    - `Normalize()`: 将向量长度变为 1（注意处理长度为 0 的情况，防止除以零崩溃）。
- [ ] **点积 (Dot Product)**: 
    - `float Dot(const Vec2& v)`: 计算 $v_1 \cdot v_2 = x_1x_2 + y_1y_2$。用于投影计算。
- [ ] **2D 叉积 (Cross Product)**: 
    - 在 2D 中，叉积返回一个标量：$x_1y_2 - y_1x_2$。用于计算转矩（Torque）和旋转。
- [ ] **距离计算**: 
    - `Distance(const Vec2& v)` 和 `DistanceSquared(const Vec2& v)`。

### C. 变换与实用工具
- [ ] **垂直向量**: 
    - `GetLeftNormal()`: 返回 `(-y, x)`。
    - `GetRightNormal()`: 返回 `(y, -x)`。
- [ ] **旋转**: 
    - `Rotate(float angle)`: 给定弧度，旋转向量。
- [ ] **零向量**: 
    - `Clear()`: 将 x, y 置 0。

## 4. 技术实现细节 (Tips)

### 浮点数精度控制
由于浮点数存在精度误差，不能直接用 `a == b`。
```cpp
const float EPSILON = 0.0001f;
inline bool Equal(float a, float b) {
    return std::abs(a - b) < EPSILON;
}
```

### 性能建议
1.  **使用 `inline`**: 所有的简单运算（如加减点积）都应该在头文件中定义为 `inline`。
2.  **Pass by Reference**: 函数参数尽量使用 `const Vec2& v` 以避免对象拷贝。
3.  **`constexpr`**: 某些常量可以使用 `constexpr` 提高编译期优化。

在二维平面中，将一个已知向量旋转一个特定角度 $\theta$ 得到新向量的过程，可以通过**三角函数几何推导**或**矩阵乘法**来实现。

## 1. 核心公式

假设已知原向量 $\vec{v} = (x, y)$，逆时针旋转角度 $\theta$（弧度）后得到新向量 $\vec{v'} = (x', y')$。

求解新坐标的公式为：


$$x' = x \cos\theta - y \sin\theta$$

$$y' = x \sin\theta + y \cos\theta$$

写成**矩阵乘法**（旋转矩阵）的形式更为直观：


$$\begin{bmatrix} x' \\ y' \end{bmatrix} = \begin{bmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{bmatrix} \begin{bmatrix} x \\ y \end{bmatrix}$$

> **注意**：如果旋转方向为顺时针，只需将 $\theta$ 变为 $-\theta$。由于 $\cos(-\theta) = \cos\theta$ 且 $\sin(-\theta) = -\sin\theta$，公式中 $\sin\theta$ 的符号对调即可。

---

## 2. 数学证明（极坐标法）

利用极坐标（Polar Coordinates）和三角函数的两角和公式，可以给出最优雅的证明。

### 第一步：用极坐标表示原向量

设原向量 $\vec{v} = (x, y)$ 的模长（长度）为 $r$，与 $x$ 轴正方向的夹角为 $\alpha$。
根据解直角三角形的几何关系，原坐标可以表示为：

1. $x = r \cos\alpha$
2. $y = r \sin\alpha$

### 第二步：表示旋转后的新向量

将向量逆时针旋转 $\theta$ 角后，新向量 $\vec{v'} = (x', y')$ 的模长依然是 $r$（旋转不改变长度），但与 $x$ 轴正方向的夹角变成了 $\alpha + \theta$。
因此，新坐标可以表示为：
3.  $x' = r \cos(\alpha + \theta)$
4.  $y' = r \sin(\alpha + \theta)$

### 第三步：应用三角两角和公式展开

利用三角函数的展开式：


$$\cos(\alpha + \theta) = \cos\alpha \cos\theta - \sin\alpha \sin\theta$$

$$\sin(\alpha + \theta) = \sin\alpha \cos\theta + \cos\alpha \sin\theta$$

将这两个展开式分别代入新坐标公式 (3) 和 (4) 中：


$$x' = r (\cos\alpha \cos\theta - \sin\alpha \sin\theta) = (r \cos\alpha) \cos\theta - (r \sin\alpha) \sin\theta$$

$$y' = r (\sin\alpha \cos\theta + \cos\alpha \sin\theta) = (r \sin\alpha) \cos\theta + (r \cos\alpha) \sin\theta$$

### 第四步：代回原坐标完成证明

将第一步中的 $x = r \cos\alpha$ 和 $y = r \sin\alpha$ 代入上式，即可得到：


$$x' = x \cos\theta - y \sin\theta$$

$$y' = y \cos\theta + x \sin\theta = x \sin\theta + y \cos\theta$$

**证毕。**

## 3. 另一种视角：基向量变换证明（线性代数法）

如果你熟悉线性代数，这个证明会更加直观：

1. 二维空间中的标准正交基为 $\vec{i} = (1, 0)$ 和 $\vec{j} = (0, 1)$。任何向量都可以表示为：$\vec{v} = x\vec{i} + y\vec{j}$。
2. 当整个空间逆时针旋转 $\theta$ 时，基向量 $\vec{i}$ 和 $\vec{j}$ 也会随之旋转：
* $\vec{i}$ 旋转 $\theta$ 后变成 $\vec{i'} = (\cos\theta, \sin\theta)$
* $\vec{j}$ 旋转 $\theta$ 后变成 $\vec{j'} = (-\sin\theta, \cos\theta)$ （因为 $\vec{j}$ 原本就比 $\vec{i}$ 超前 $90^\circ$）


3. 根据线性变换的性质，旋转后的新向量 $\vec{v'}$ 可以由旋转后的基向量线性组合得到：

$$\vec{v'} = x\vec{i'} + y\vec{j'} = x\begin{bmatrix} \cos\theta \\ \sin\theta \end{bmatrix} + y\begin{bmatrix} -\sin\theta \\ \cos\theta \end{bmatrix} = \begin{bmatrix} x\cos\theta - y\sin\theta \\ x\sin\theta + y\cos\theta \end{bmatrix}$$



这直接推导出了旋转矩阵。
