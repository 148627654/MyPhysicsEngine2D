# V2-Day 2: 启发式插入算法 (InsertLeaf)

## 1. 目标
实现动态 AABB 树最核心的构建逻辑：**将叶子节点插入树中**。我们要确保新加入的物体能找到最合适的“邻居”，并自动生成层级结构，同时通过回溯更新确保父节点的 AABB 永远包含子节点。

## 2. 核心概念

### A. 寻找最佳兄弟 (Search for Best Sibling)
当你把一个新叶子节点（物体 A）放入树中时，不能随便找个地方挂。你需要寻找一个现有的节点（节点 B），使得 A 和 B 合并后的**代价最小**。
*   **代价计算 (Cost Function)**：在 2D 中，通常使用**周长 (Perimeter)**。如果 A 和 B 合并后的 AABB 周长增加越小，说明它们在空间上越接近，查询效率越高。

### B. 内部节点 (Internal Nodes)
动态 AABB 树是一棵满二叉树。当你决定把叶子 A 挂在节点 B 旁边时：
1.  从内存池分配一个新的**内部节点 P**。
2.  P 成为 A 和 B 的共同父节点。
3.  P 的 AABB 等于 `Union(A.aabb, B.aabb)`。

### C. 向上回溯 (Bottom-up Update)
插入完成后，从新节点的父节点开始，沿着 `parent` 指针一路向上直到根节点（Root），重新计算每一层内部节点的 AABB。这保证了根节点的包围盒能盖住全世界。

## 4. 技术实现细节 (Tips)

### 1. 启发式搜索简化逻辑
为了 Day 2 的稳定性，我们不需要实现极其复杂的 SAH 搜索，只需遵循：
*   如果当前节点是叶子，计算 `Union(Leaf, New).Area`。
*   如果当前节点是内部节点，计算 `Union(Internal, New).Area - Internal.Area`（即新增代价）。

### 2. 满二叉树的父子关系
当你把节点 `N` 插入到节点 `S`（Sibling）的位置时：
*   原本 `S` 的父节点 `OldParent` 现在应该指向新分配的内部节点 `P`。
*   `P` 的两个孩子是 `S` 和 `N`。
*   `S` 和 `N` 的父节点都设为 `P`。

void DynamicTree::InsertLeaf(int32_t leaf) {
    if (m_root == -1) {
        m_root = leaf;
        m_nodes[m_root].parent = -1;
        return;
    }

    // 1. 寻找最佳兄弟 (这里可以用最简单的寻找逻辑)
    AABB leafAABB = m_nodes[leaf].aabb;
    int32_t index = m_root;
    while (!m_nodes[index].IsLeaf()) {
        // 启发式：选择周长增加最小的孩子进入 (Day 3 会进一步优化)
        int32_t child1 = m_nodes[index].leftChild;
        int32_t child2 = m_nodes[index].rightChild;
        // ... 代价计算逻辑 ...
        index = (cost1 < cost2) ? child1 : child2;
    }
    int32_t sibling = index;

    // 2. 创建新的内部节点作为父节点
    int32_t oldParent = m_nodes[sibling].parent;
    int32_t newParent = AllocateNode();
    m_nodes[newParent].parent = oldParent;
    m_nodes[newParent].aabb = AABB::Union(leafAABB, m_nodes[sibling].aabb);
    m_nodes[newParent].height = m_nodes[sibling].height + 1;

    // 3. 处理连接关系 (这是最容易出错的地方，请画图对照)
    if (oldParent != -1) {
        if (m_nodes[oldParent].leftChild == sibling) m_nodes[oldParent].leftChild = newParent;
        else m_nodes[oldParent].rightChild = newParent;
    } else {
        m_root = newParent;
    }
    m_nodes[newParent].leftChild = sibling;
    m_nodes[newParent].rightChild = leaf;
    m_nodes[sibling].parent = newParent;
    m_nodes[leaf].parent = newParent;

    // 4. 向上回溯同步 AABB
    int32_t p = m_nodes[leaf].parent;
    while (p != -1) {
        // 更新 AABB 和 Height
        // ...
        p = m_nodes[p].parent;
    }
}
