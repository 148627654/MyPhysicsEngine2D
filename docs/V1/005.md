# Day 05: 分离轴定理 (SAT) 与矩形碰撞检测

## 1. 目标
实现 `Box` (矩形) 与 `Box` 之间的碰撞检测，能够准确识别旋转状态下的矩形是否接触，并提取出**最小穿透深度**和**碰撞法线**。

## 2. 核心概念：SAT (分离轴定理)
**基本原理**：如果两个凸多边形没有碰撞，那么一定存在一条直线，使得这两个多边形在那条直线上的投影不重叠。
- 对于矩形，我们只需要检查 **4条轴**：矩形 A 的两条局部坐标轴（长和宽的方向）以及矩形 B 的两条局部坐标轴。
- 如果在这 4 条轴上，投影都有重叠，那么碰撞发生。
- **穿透深度** = 4 条轴中重叠量最小的那个值。
- **碰撞法线** = 重叠量最小的那条轴的方向。

## 4. 技术实现细节 (Tips)

### 1. 如何获取矩形的轴？
矩形的轴其实就是它旋转后的局部坐标系向量。
如果角度为 $\theta$：
- 轴 1 (X轴): `(cosθ, sinθ)`
- 轴 2 (Y轴): `(-sinθ, cosθ)`

### 2. 投影计算公式
顶点 $\vec{V}$ 在单位向量轴 $\vec{L}$ 上的投影标量值为：$p = \vec{V} \cdot \vec{L}$。
遍历 4 个顶点，记录最小值和最大值。

### 3. MTV (Minimum Translation Vector)
物理引擎总是希望通过“代价最小”的方式把物体推开。所以 SAT 选取的轴必须是**重叠量最小**的那一根。

### 💡 核心逻辑预览 (伪代码)

```cpp
bool Collision::BoxVsBox(Manifold* m, Body* a, Body* b) {
    float minOverlap = FLT_MAX;
    Vector2 bestAxis;

    // 1. 获取 A 和 B 的所有潜在分离轴 (共4条)
    std::vector<Vector2> axes = { a->getAxisX(), a->getAxisY(), b->getAxisX(), b->getAxisY() };

    for (Vector2& axis : axes) {
        // 2. 将 A 和 B 的顶点投影到轴上
        auto projA = a->getProjection(axis); // 返回 {min, max}
        auto projB = b->getProjection(axis);

        // 3. 检查重叠
        float overlap = GetOverlap(projA, projB);
        if (overlap <= 0) return false; // 只要有一条轴不重叠，立即返回 false

        // 4. 记录最小重叠
        if (overlap < minOverlap) {
            minOverlap = overlap;
            bestAxis = axis;
        }
    }

    // 5. 填充 Manifold
    m->penetration = minOverlap;
    m->normal = bestAxis;
    // ... 确保法线方向正确 ...
    return true;
}
```
