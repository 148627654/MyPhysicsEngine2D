# Day 09: 线性冲量解算与弹性碰撞

## 1. 目标
根据碰撞流形（Manifold）提供的信息，计算并应用**线性冲量 (Linear Impulse)**。实现物体的物理反弹，确保碰撞后的运动符合**动量守恒定律**。

## 2. 核心概念：线性冲量 (Impulse)
由于碰撞发生在极短的时间内，我们不使用力（Force），而是使用速度的突然变化，即**冲量 $J$**。

*   **牛顿恢复定律**：碰撞后的相对速度 = $-e \times$ 碰撞前的相对速度。
    *   $e = 0$：完全塑性碰撞（粘在一起）。
    *   $e = 1$：完全弹性碰撞（完美反弹，无能量损失）。
*   **冲量公式**：
    $$j = \frac{-(1+e)(\vec{v}_{rel} \cdot \vec{n})}{\frac{1}{m_A} + \frac{1}{m_B}}$$
    其中 $\vec{v}_{rel}$ 是 A 和 B 的相对速度，$\vec{n}$ 是碰撞法线。

## 3. 必须实现的功能清单

### A. 增强 Body 类 (Dynamics 模块)
- [ ] **恢复系数 (Restitution)**：为 `Body` 或 `Shape` 增加 `restitution` 属性（建议范围 0.2 ~ 0.8）。
- [ ] **应用冲量方法**：实现 `ApplyImpulse(Vector2 impulse)`，直接修改 `velocity`。
    *   $\vec{v} = \vec{v} + \vec{J} \times invMass$

### B. 实现冲量解算器 (Impulse Solver)
- [ ] **相对速度**：$\vec{v}_{rel} = \vec{v}_B - \vec{v}_A$。
- [ ] **法向速度**：$v_{normal} = \vec{v}_{rel} \cdot \vec{n}$。
- [ ] **判定逻辑**：如果 $v_{normal} > 0$，说明物体正在分离，**不要**应用冲量。
- [ ] **计算标量冲量 $j$**：使用上面的公式计算 $j$。
- [ ] **应用向量冲量**：
    *   $\vec{J} = j \cdot \vec{n}$
    *   `A->ApplyImpulse(-J)`
    *   `B->ApplyImpulse(J)`

## 4. 技术实现细节 (Tips)

### 1. 处理静态物体 (Infinite Mass)
你的代码已经存储了 `invMass`。如果物体是地面，其 `invMass = 0`。
在公式中，分母 $1/m_A + 1/m_B$ 会自动处理这种情况：
*   如果两个物体都是 1kg，分母是 2。
*   如果 A 撞击地面，分母是 $1 + 0 = 1$。
*   如果两个都是地面，分母是 0（这种情况理论上不会发生，可以在代码里加个保险）。

### 2. 什么是好的 $e$ 值？
给球体设置 `restitution = 0.5`，给地面设置 `restitution = 0.5`。
混合恢复系数公式通常采用：`e = std::min(A->e, B->e)`。

### 3. 注意法线方向
确保你的 `j` 计算中，法线是从 A 指向 B。如果反了，物体会像磁铁一样吸在一起。

### 💡 核心逻辑预览 (Solver.cpp)

```cpp
void ResolveCollision(Manifold& m) {
    Body* A = m.bodyA;
    Body* B = m.bodyB;

    // 1. 计算相对速度
    Vector2 relativeVel = B->GetVelocity() - A->GetVelocity();

    // 2. 计算法向相对速度
    float velAlongNormal = relativeVel.Dot(m.normal);

    // 3. 如果速度是分离的，不处理
    if (velAlongNormal > 0) return;

    // 4. 计算恢复系数 e
    float e = std::min(A->restitution, B->restitution);

    // 5. 计算冲量标量 j
    float j = -(1 + e) * velAlongNormal;
    j /= (A->invMass + B->invMass);

    // 6. 应用冲量
    Vector2 impulse = m.normal * j;
    A->velocity -= impulse * A->invMass;
    B->velocity += impulse * B->invMass;
}
```
