# Day 06: 混合形状碰撞 (Circle vs. Box)

## 1. 目标
实现 `Circle` 与 `Box` 之间的碰撞检测逻辑。要求在矩形处于**任意旋转角度**时，都能准确计算出碰撞法线、穿透深度和接触点。

## 2. 核心概念：最近点法 (Nearest Point on Box)
由于圆有无限多条分离轴，使用 SAT 处理圆与矩形虽然可行但不够高效。最通用的做法是：**在矩形上找到一个距离圆心最近的点。**

*   **局部空间化**：将圆心转换到矩形的局部坐标系（让矩形变回轴对齐、中心在原点的状态）。
*   **钳位 (Clamping)**：通过限制坐标范围，在矩形边界上锁定最近点。
*   **距离判定**：如果“圆心”到“最近点”的距离小于“圆的半径”，则发生碰撞。

---

## 3. 必须实现的功能清单

### A. 局部坐标映射 (Local Space Mapping)
- [ ] **逆向变换**：计算圆心相对于矩形中心的位置，并对其进行**反向旋转**（旋转角度为 `-box.rotation`）。
    *   `localCirclePos = (worldCirclePos - worldBoxPos).Rotate(-boxAngle)`

### B. 寻找最近点 (Clamping)
- [ ] **锁定坐标**：将 `localCirclePos` 的 X 和 Y 坐标分别钳位在矩形的半宽（halfWidth）和半高（halfHeight）范围内。
    *   `closestX = Clamp(localX, -hw, hw)`
    *   `closestY = Clamp(localY, -hh, hh)`

### C. 碰撞判定与流形填充
- [ ] **距离计算**：计算 `localCirclePos` 与 `closestPoint` 的距离 $d$。
- [ ] **外部碰撞**：如果 $d > 0$ 且 $d < radius$，碰撞发生。
    *   法线 = `(localCirclePos - closestPoint).Normalize()`
- [ ] **内部碰撞（圆心在矩形内）**：如果 $localCirclePos == closestPoint$（即 $d=0$），说明圆心在矩形内。此时需找到距离圆心最近的一条边，将圆推出去。
- [ ] **空间回转**：将计算出的局部法线 `localNormal` **正向旋转**回世界坐标系。

---

## 4. 技术实现细节 (Tips)

### 1. 为什么一定要转到局部空间？
在世界空间中，如果矩形旋转了，求最近点的数学公式会涉及复杂的三角函数。而在局部空间中，矩形永远是“正”的，最近点只需要两次简单的 `std::max / std::min`。

### 2. 修正法线方向
物理引擎规范：法线应从 `bodyA` 指向 `bodyB`。
如果你实现的函数是 `CircleVsBox(Manifold* m, Body* circle, Body* box)`，那么：
*   `m->normal` 应从圆指向矩形。
*   如果圆心在外部，局部法线是 `closestPoint - localCirclePos`。

### 3. 处理圆心在内的特殊情况
当圆心在矩形内时，`Clamping` 得到的点就是圆心自己。
此时，你需要比较 `localCirclePos` 到矩形四条边的距离，选最短的那条。例如：
`distToRight = hw - localCircleX;`
如果 `distToRight` 是最小的，那么局部法线就是 `(1, 0)`，穿透深度是 `radius + distToRight`。

```cpp
bool Collision::CircleVsBox(Manifold* m, Body* circleBody, Body* boxBody) {
    Circle* circle = static_cast<Circle*>(circleBody->GetShape());
    Box* box = static_cast<Box*>(boxBody->GetShape());

    // 1. 世界坐标转 Box 局部坐标
    Vector2 relativePos = circleBody->GetPosition() - boxBody->GetPosition();
    Vector2 localCirclePos = relativePos.Rotate(-boxBody->GetRotation());

    // 2. 寻点
    float hw = box->getHalfWidth();
    float hh = box->getHalfHeight();
    Vector2 closestPoint(
        std::max(-hw, std::min(localCirclePos.getX(), hw)),
        std::max(-hh, std::min(localCirclePos.getY(), hh))
    );

    // 3. 计算距离与判定
    Vector2 distVec = localCirclePos - closestPoint;
    float distSq = distVec.LengthSquared();
    float r = circle->getRadius();

    if (distSq > r * r && !Inside(localCirclePos, hw, hh)) return false;

    // 4. 填充 Manifold (涉及局部转世界法线)
    // ...
    return true;
}
```
